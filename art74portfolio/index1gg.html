Advanced DIYPTS


This DIY photoshop was made by using Processing and P5js. A completed code can be found under these examples. My final project is about developing more tools, in additional to the previous DIY Photoshop. The examples below are actual works of my friends using my code. See below for instructions.

If you do not have Processing with p5js, you can use this online website here: https://editor.p5js.org/

Instruction:
Please press "[" or "]" key and start at thee tool has 3 basic colors for drawing lines (Red, Green, and Blue).
Press "r", "g", "b" for thin lines. Press "R", "G", "B" for thick lines (caplock (or hold shift) + r, g, b).
Press "," for a yellow block, or press "." for a white block.
Press ";" for a white circle, or press "/" for a brown circle.
Press key number from 1 to 4 to select different symbols.
Press key number from 5 to 7 to select different glitch tools.
