Advanced DIYPTS


This DIY photoshop was made by using Processing and P5js. A completed code can be found under these examples. My final project is about developing more tools, in additional to the previous DIY Photoshop. The examples below are actual works of my friends using my code. See below for more instructions.

If you do not have Processing with p5js, you can use this online website here: https://editor.p5js.org/
Note: this website includes an outside source for the p5.js library. Please allow the access to that website from your browser in order to use the live canvas below. The website is safe.

P/s: For future purposes, here is how you include a live canvas from a js file to your website.
From your p5.js file, add to your setup() function:
var canvas = createCanvas(800, 600); //so createCanvas cammand is now a variable
canvas.parent('parentid'); //so you can add this id to your html code

Add to your html head:
script src="https://cdnjs.cloudflare.com/ajax/libs/p5.js/0.5.6/p5.js" //p5js library: to convert js to javascript
script src="yourjs" //the name of your js file goes here (add your js file to dreamweaver first (or something similar))

Add to your html body:
div id="parentid" //add this line to where you want your canvas goes

Functions of the live canvas:
Please press "[" or "]" key and start at thee tool has 3 basic colors for drawing lines (Red, Green, and Blue).
Press "r", "g", "b" for thin lines. Press "R", "G", "B" for thick lines (caplock (or hold shift) + r, g, b).
Press "," for a yellow block, or press "." for a white block.
Press ";" for a white circle, or press "/" for a brown circle.
Press key number from 1 to 4 to select different symbols.
Press key number from 5 to 7 to select different glitch tools.
