Advanced DIYPTS


This DIY photoshop was made by using Processing and P5js. A completed code can be found under these examples. See below for instruction.

Please press "[" or "]" key and start at the upper left corner to import a picture.
Press "x" to erase sketch, or press "s" to save sketch.
The tool has 3 basic colors for drawing lines (Red, Green, and Blue).
Press "r", "g", "b" for thin line. Press "R", "G", "B" for thick line (caplock (or hold shift) + r, g, b).
Press "," for a yellow block, or press "." for a white block.
Press ";" for a white circle, or press "/" for a brown circle.
Press key number from 1 to 4 to select different symbols.
Press key number from 5 to 7 to select different glitch tools.
